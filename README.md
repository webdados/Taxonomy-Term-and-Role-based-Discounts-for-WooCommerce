# Taxonomy/Term and Role based Discounts for WooCommerce
"Taxonomy/Term based Discounts for WooCommerce" lets you configure discount/pricing rules for products based on any product taxonomy terms and WordPress user roles

This is the free plugin available at https://wordpress.org/plugins/taxonomy-discounts-woocommerce/

There is a PRO add-on available at https://nakedcatplugins.com/product/taxonomy-term-and-role-based-discounts-for-woocommerce-pro-add-on/
